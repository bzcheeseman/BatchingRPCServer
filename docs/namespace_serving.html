Detailed Description

Top level namespace for this project.

Defines the namespace within which all the code in this project resides. Protobuf generates code under this namespace so for uniformity everything lives under this namespace. No subdivisions exist as of 26/12/2017.

enum Serving::ReturnCodes

Possible return codes for a Servable object.

These are the return codes that each Servable object may return. Some codes will only be returned by certain functions.

Enumerator
OK 

Function exited normally.

NEED_BIND_CALL 

Bind was not called on the Servable, typically done before construction of TBServer. See Servable::Bind(BindArgs&).

SHAPE_INCORRECT 

The shape of the input was incorrect, check the shape and retry.

NEXT_BATCH 

The current batch is too full to process the request, retry.

BATCH_TOO_LARGE 

The request's size is too large for the Servable's set batch size, caller should subdivide and try again or increase the batch size.

NO_SUITABLE_BIND_ARGS 

Bind has failed because the Servable was unable to cast the BindArgs instance to a usable type.
