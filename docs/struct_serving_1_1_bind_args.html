Detailed Description

A virtual wrapper for arguments used to bind a Servable.

All servables will require different arguments to bind, but they must create a subclass of this object and then use a dynamic cast to cast it to the appropriate object. See an example in MXNetServable.hpp - Serving::RawBindArgs and Serving::FileBindArgs.
